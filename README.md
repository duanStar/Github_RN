## 环境配置
[环境搭建](https://www.reactnative.cn/docs/environment-setup)

## 如何运行
1. 在项目根目录执行`npm install`或者`yarn install`
2. 切换到ios目录执行`pod setup`来拉取最新pod库
3. 然后运行`yarn android`或`yarn ios`
